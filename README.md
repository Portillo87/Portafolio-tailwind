# Portafolio Tailwind - Proyecto Estudiante

Versión de ejemplo del portafolio para la asignatura "Diseño Web Adaptable".
Incluye:
- index.html (Home con hero, carousel y servicios)
- portafolio.html (galería con modals accesibles)
- sobre-mi.html (timeline y barras de progreso)
- contacto.html (formulario con validación y alert ARIA)
- assets/ (imágenes placeholder)

Instrucciones rápidas:
1. Descomprime y abre `index.html` en tu navegador para ver el sitio localmente.
2. Para publicar en GitHub Pages:
   - Crea un nuevo repositorio
   - Añade los archivos y hace commit
   - Subir a la rama main y activar GitHub Pages desde Settings -> Pages -> Deploy from branch (main)

Accesibilidad:
- Roles ARIA usados en navegación y modals.
- Formulario incluye role="alert" para feedback.
- Revisa contraste y usa Wave/WebAIM para validación adicional.

Bibliografía:
- MDN - ARIA Roles: https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Reference/Roles
- Tailwind CSS: https://tailwindcss.com/
