Proyecto 1 — Landing responsiva

Landing page creada con Tailwind CSS. Incluye sección hero, cards de productos, formulario de contacto y footer. Se diseñó mobile-first y se realizaron pruebas en distintos breakpoints.

Tecnologías

  • HTML5 y CSS3
  • Tailwind CSS (utilidades y breakpoints)
  • JavaScript para validación mínima

Accesibilidad aplicada

  • Etiquetas semánticas (header, main, article, footer)
  • Alt descriptivos en imágenes
  • Roles ARIA en navegación y alertas
